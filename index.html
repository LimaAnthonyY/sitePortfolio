  • Encerramento do Primeiro Módulo de Engenharia de Dados no Mackenzie

    Durante este primeiro módulo do curso de Engenharia de Dados no Mackenzie, apresentamos nosso modelo de previsão de consumo elétrico baseado em sazonalidade. Utilizamos dados fornecidos pela PJM Interconnection, uma organização de transmissão regional nos Estados Unidos que faz parte da rede Eastern Interconnection. Este modelo foi desenvolvido utilizando a biblioteca Prophet, proporcionando uma abordagem robusta e eficaz para prever padrões de consumo elétrico com precisão, veja mais aqui no Linkedin.

    Este projeto envolve a criação de um dashboard interativo no Power BI, que utiliza três conjuntos de dados distintos sobre os Pokémons. Cada conjunto de dados é manipulado para fornecer informações específicas e relevantes para a análise dos status dos Pokémon.

  • Explorando o Universo Pokémon com PySpark!
